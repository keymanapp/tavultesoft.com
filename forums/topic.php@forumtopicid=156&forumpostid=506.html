# RE: Keyman does not work properly in Vista   2007-07-30 15:45:05.920
Marc Durdin
Tavultesoft Staff
If you use the Light edition of Keyman Desktop, it does not support the Language Association option. The Professional edition includes the "Languages" tab which lets you select the keyboard to associate with any language on your system.

A related note: We have a documented issue currently with Vista that will be addressed in the next update of Keyman Desktop. This issue is that the current associated languages settings do not display in Keyman Configuration when you start it up under Windows Vista. However, you can set the language associations in Keyman Configuration and click Save, and they will be applied to Keyman Desktop until the next time you run Keyman Configuration.

I have just tested this here with Vista and Word 2007 and it all worked without issues with version 240. Can you check the following for me:

1. You need to associate the Greek keyboard with the Greek language in the Languages tab in Keyman Configuration.

2. To use the Keyman keyboard in the way you describe, you should switch the language to Greek and the Keyman keyboard will switch as well.
