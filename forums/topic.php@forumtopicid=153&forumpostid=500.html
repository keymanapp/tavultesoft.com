By default, Keyman keyboards use a fixed layout, and ignore the underlying layout (for convenience, this fixed layout is based on US English for a number of reasons). In order to base your layout on the underlying system layout, you need to use the "mnemonic layout" feature. To turn on mnemonic layout for your keyboard, add the following header statement in your .kmn source file, recompile and reinstall it:

:
store(&mnemoniclayout) '1'
