# RE: Installing a keyboard in Developer fails with “Keyman Desktop is not installed”   2017-03-01 21:11:10.130
Marc Durdin
Tavultesoft Staff
Yes, Keyman Desktop 9 and Keyman Developer 8 don't really work well together. You should try to use the same version for both products -- Keyman Developer 9 is a free version (in beta, but still stable) -- get it from http://tavultesoft.com/beta/
