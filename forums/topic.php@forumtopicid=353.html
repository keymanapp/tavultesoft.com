Tavultesoft Staff
Hi Tano Fotang,

Thanks for your feedback. We have an development issue open regarding the incorrect link on the purchase screen, and this will be corrected in the next beta. In the meantime, you can visit https://secure.tavultesoft.com/buy/90/ to purchase a version 9.0 license.

Regarding the 8.0 crash, this happened because there were still files from the version 9.0 beta left on your machine; you would need to uninstall Keyman Desktop 9 and Keyman Developer 9, restart Windows. You should then delete the folders "C:\Program Files (x86)\Tavultesoft" and "C:\Program Files (x86)\Common Files\Tavultesoft", if they exist, and then install version 8. This happens because there were files that were not removed by the version 9.0 beta installer (although these are mostly addressed with the latest beta).
