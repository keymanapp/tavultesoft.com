# RE: Keyman desktop won't show installed keyboard   2014-12-19 03:01:02.663
Steven White
What I meant by the main window is what appears when I click the Keyman icon in the notification area. In the list of keyboards, I only saw "English (United States)".

I have now gotten Keyman desktop to work. I had folders for older Keyman versions in Program files (x86)\Common files\Tavultesoft, and there were also references in my registry, in HKLM\SOFTWARE\Wow6432Node\Tavultesoft.

When I cleared all those out, then did a "repair" from the Keyman Desktop 9.0 installer, the keyboards started displaying.
