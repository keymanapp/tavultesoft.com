Andrew C.
Keyboard Designer
There are three possibilities that i can see:

1) Your web site builder is not Unicode based;

2) Your web site builder is not currently using Unicode and needs to be set to use Unicode (in its options or preferences), as Marc suggested; or

3) Your web site builder is Unicode based by uses the old Windows 95 internationalization model, and can only support languages that are available via single Microsoft codepages.

If your web site builder cannot support Unicode, you will need ot use different software or type up the text in a text editor and then convert the Unicode text to numerical character references (NCRs) and paste the NCRs into the web site builder.
