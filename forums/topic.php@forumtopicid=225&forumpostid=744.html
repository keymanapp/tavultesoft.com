You can uninstall a keyboard layout by clicking on the options button beside that layout in Keyman Configuration's Keyboard Layout tab.

You can disable a keyboard layout without uninstalling it by clicking on the checkbox to the left of the options button in Keyman Configuration's Keyboard Layout tab.

# RE: remove languages from configuration   2009-04-02 09:37:37.793
Tavultesoft
The languages listed in the Keyman Configuration Languages tab are Windows languages.
To remove them from Vista (see XP below), from the Control Panel, select 'Change keyboards...' under 'Clock, Language and Region' or select the 'Keyboards and Languages tab' in 'Regional and Language Options'. Then click 'Change Keyboards' and select the Windows languages and keyboards you wish to remove.
To remove them from XP, from the Control Panel, select the 'Languages tab' in 'Regional and Language Options'. Click 'Details' and select the Windows languages and keyboards you wish to remove.
