Tavultesoft Staff
The Font Helper is not currently available in Keyman Developer but we do plan to integrate it into the Character Map in Keyman Developer in a future update.

You can find a character in character map simply by clicking on the character or U+xxxx code in the keyboard source. When you do this, the Character Map will automatically jump to that character, unless:

    You have filtered the character map to exclude that character, or
    You have switched off the option "Find character under cursor automatically"


A function to show all characters in a keyboard would be helpful and we will consider it for a future update of Keyman Developer, perhaps related to the Font Helper.
