# RE: Support for Extended shift flags   2017-01-10 00:42:36.363
Marc Durdin
Tavultesoft Staff
Until recently, most browsers did not distinguish between left and right ctrl/alt, so there was no way to detect them from Javascript. I haven't researched the current support level for this feature in browsers. You could submit an issue at https://github.com/tavultesoft/keymanweb/issues to request that we add support for these flags in KeymanWeb on those browsers that support l/r ctrl/alt.
# RE: Support for Extended shift flags   2017-01-10 00:46:59.637
Marc Durdin
Tavultesoft Staff
In terms of working around the issue in your keyboard, you will need to create rules that disambiguate desktop and web builds:, for example:
:
$keymanonly: + [LCTRL K_W] > 'LCtrl W'
$keymanweb: + [CTRL K_W] > 'Control W'
