# RE: IPA keyboard not showing up as option   2007-09-07 09:35:17.097
Marc Durdin
Tavultesoft Staff
It sounds like there is a conflict with another utility application on the system. Some poorly written utility applications can conflict with Keyman 6.0 because they don't follow Microsoft's standards and requirements. Version 6.2.183.0 and version 7.0 of Keyman Desktop both include special code to detect these applications and work around the issue. However, version 6.0 does not include this code. I recommend upgrading to 6.2 (available to licenced users of version 6, 6.1 and 6.2) or 7.0 (trial version downloadable from http://www.tavultesoft.com/70/download.php).

An alternative is to review the utility applications running on your system to try and find the one that is conflicting. There are various system information tools available that will show you a list of applications that start with Windows. One that comes with windows is "msconfig". To access msconfig, click Start Menu, Run, type "msconfig" and press Enter. Look at the Startup tab to see the list of programs that start with Windows.
