Hi Leonardo,

It is certainly possible. However, as you discovered, the visual editor does not display the 2 additional keys. There are two steps to adding the rules for the extra keys to your keyboard.

1. Use Tools | Virtual Key Identifier to discover the key codes for those extra keys.

2. In the Source tab for your keyboard, add rules using those key codes, e.g. for the 102nd key on a European keyboard it would be like this:

+ [K_oE2] > '\'
