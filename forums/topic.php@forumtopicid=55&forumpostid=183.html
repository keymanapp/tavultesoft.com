Hi Maziar,

I suggest you try installing a right-to-left language (Farsi by default in Windows) in Windows Control Panel Regional Settings (Languages tab). Once installed, associate it with the Persian keyboard in Keyman Configuration (Languages tab).

Then when you enable that language using the Windows Language bar and the Persian keyboard, you can type right-to-left and the parentheses should work fine.

It's unfortunately rather confusing, but once it's working it works well.

Tavultesoft keyman keyboards work best under Windows ME or 98. Under Windows XP they also work, but here Farsi keyboard can also be loaded in Control Pannel, Regional Settings. There you can also alot keystroke combinations (Ctrl or Left Alt + ...)to various keyboards.
Those who write in two languages simultaneously, it is advisable to alot keyboard-macro commands for switching between Arabic(Farsi)and Latin keyboards. You can handle this klicking Extras/Preferences(in German Anpassen)/keyboard (Tastatur)/All commands. Then choose Run (Laufumschalten) and give Alt + L as keystrokes to change from Latin to Arabic and the other way round.
