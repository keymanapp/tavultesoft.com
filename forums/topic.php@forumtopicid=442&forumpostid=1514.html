# A request to be able to monitor the value of selected Stores.   2016-03-01 08:00:49.180
br Julian Griffin OSL
Most Stores are constants so don't need to be monitored. But with the advent of the Set command, some Stores can change. So it would therefore be very nice if one where able to monitor the value of a selection of these Stores in the Debugger Status window.

   Although, the Elements tab does show the value of a Store when it is in use (setting or testing), I feel it would still be useful if there was a tab in the Debugger Status that either listed all Stores that could change or allow the selection of Stores from a dropdown list.

   And just to be cheeky, I would also ask that the list of selected Stores be retained when Keyman Developer 9 is closed and reinstated the next time Keyman Developer 9 is re-started.
