You can modify just the keys you need if you use the mnemonic layout approach:

:
store&#40;&amp;mnemoniclayout&#41; '1'


This tells Keyman to follow the position of the characters on the key caps rather than ignoring them (which is a "positional" or "typewriter" layout). An added advantage of a mnemonic layout is that the Keyman keyboard will also work without modification on US, German, French etc keyboards.

You must edit mnemonic layouts in the source editor, not the keyboard wizard. You can read more about mnemonic layouts here:

http://www.tavultesoft.com/keymandev/documentation/help/reference/mnemonic.php

You may also find the documentation on virtual character keys useful:

http://www.tavultesoft.com/keymandev/documentation/help/reference/_struct_vkeys.php
