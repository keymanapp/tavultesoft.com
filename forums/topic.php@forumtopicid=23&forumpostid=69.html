The Keyman icon is a K on a key, and is displayed in the bottom right of the screen. You can confirm that Keyman is installed correctly by navigating to the Start Menu, and looking for the Tavultesoft Keyman folder in the Start Menu. Hi Glennie,

If you installed the package "Download "IPA93_KM5.kmp", at http://scripts.sil.org/cms/scripts/page.php?site_id=nrsi&amp;item_id=encore-ipa-download then there will be a document available from the Start Menu -&gt; All Programs -&gt; IPA93 1.2 (FF) -&gt; IPA93 Keyboard Layout.

The basic letters in the IPA font are modified by typing a special key after the initial character, the primary modifiers being "&lt;", "&gt;" and "=". Full details are in the document I mentioned above.
