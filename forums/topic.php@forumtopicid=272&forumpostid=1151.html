# RE: Configuring Keyman 6.2 with Windows 7   2011-07-27 00:02:05.490
Arun Sarkar
Hello Clare :)
I don't know about windows 7, But I can run Keyman 6.2 on vista.
Go to
1)Program Files
2) Go to Tavultesoft folder
3) Go to Keyman Folder
4) Select Keyman.exe
5) Right click on it
6) Properties -- Compatibility mode
7) Select Windows XP SP 2 -- apply -- ok

It works for my Vista Home Basic. As Vista having a XP mode & If your Windows 7 (most probably Premium Version) have this XP feature then it may works.

Regards
Arun
