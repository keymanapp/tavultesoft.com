Tavultesoft Staff
Which versions of Keyman and Keyman Developer do you have installed?

It sounds like you are trying to compile a redistributable installer package. In Keyman Developer 6.x, this requires a valid installation of Keyman, as it uses the Keyman installation when building the installer.

This message means that that the Keyman installation is not being recognized by Keyman Developer. The easiest way to fix the issue is to install Keyman again (no need to uninstall first). It can also occur if you have different versions of Keyman and Keyman Developer installed, if you are using Keyman Developer 6.0, 6.1 or 6.2.

You can also check the [root path] registry setting for Keyman in HKLM\Software\Tavultesoft\Keyman\6.0. If this entry is missing, or the path it refers to does not exist, this issue can occur.
