# RE: Keyman desktop 9 -- install keyboard error.   2014-12-16 07:00:14.823

Update -- I figured out the problem. Keyman won't load a KMX file that is located on my network drive. When I copy the KMX file to my local hard drive, I can install it.
# RE: Keyman desktop 9 -- install keyboard error.   2014-12-16 08:00:19.530
Marc Durdin
Tavultesoft Staff
Glad to hear you figured out the problem. This is a common issue for installers and the like: when you elevate to install, the elevated user token doesn't have access to the same resources as the standard user token. Even Explorer suffers from this problem in some configurations -- try copying a file from your network to C:\Program Files, for example.

And as you found, the simplest resolution is to copy the file to your local drive before installing.
