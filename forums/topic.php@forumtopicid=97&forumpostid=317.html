Hi Zak,

I believe this problem is occurring because the Classical Greek keyboard is associated with the Windows Greek keyboard layout.

To solve this problem:
1) Open Keyman Configuration.
2) Select the Languages tab.
3) Where Greek is listed in the Language column, select (none) in the Keyman Keyboard column.
4) Click OK.

This should remove the automatic association of Classical Greek with the Windows Greek layout.
