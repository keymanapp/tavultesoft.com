Marc Durdin
Tavultesoft Staff
Thank you for the report. We have reviewed the diagnostic report and identified a problem in the Keyman Desktop RichEdit Addin which we will resolve in the next update of Keyman Desktop.

As a workaround you can temporarily uninstall the RichEdit Addin with the following steps:

1. Close Keyman Desktop if it is running
2. Open Control Panel, Add/Remove Programs
3. Locate Tavultesoft Keyman Desktop Professional 7.0 and click Change
4. Click Next, Change, expand the [+] next to the "Addins" entry, click the "RichEdit Addin" entry, and select "Entire feature will be unavailable"
5. Click Next, Change
