Hi Vivek,

From what I can tell by using the keyboard, the only sure way to get that character appears to be by entering the following sequence:

r + 200D + y + a

Where 200D is the Unicode zero width joiner character. To enter this character, you can use the Keyman hexkeys method available under Special Keys in Keyman Configuration. By default, hold down Right-Ctrl + Shift and type 200d.

In Notepad you can right-click and select Insert Unicode Control Character and choose the zero width joiner from there.
