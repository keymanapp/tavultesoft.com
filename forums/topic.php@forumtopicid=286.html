The first things to rule out are:

* the text layout/font rendering system supports rendering combining diacritics; and
* an opentype font suitable for Nuer is being used to display text. Usual safe bets are the SIL fonts: Charis SIL, Doulos SIL and the recently released Gentium Plus. There are a number of other fonts that would also be suitable.

if you are having trouble with (?¨?_), then i'll assume you are also having trouble with (?_¨).

Could you indicate which operating system, web browser (and version) and which fonts you are using. That will narrow the problem down to either a template/theme problem or an input problem.
