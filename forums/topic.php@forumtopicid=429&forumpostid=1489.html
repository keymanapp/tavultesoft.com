# RE: Transparent icons   2016-02-22 07:37:28.320
Marc Durdin
Tavultesoft Staff
The icon should not appear as having a white background in the notification area but will have a white background in the menu, for example.

So I'm not sure what is going on - transparent is selected in the colour picker by clicking the colour box with an [X] in it. The PNG import will probably not support transparency as PNG files support full alpha transparency, whereas the Keyman Developer icon editor only supports simple transparency.

You can edit the .ico file for a keyboard in any icon editor tool and add full alpha transparent versions and multiple sizes for High DPI screens.
