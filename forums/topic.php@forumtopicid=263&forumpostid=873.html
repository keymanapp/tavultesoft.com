That is a weird problem! Just h, c and v? Two things to try:

1. It may be that some software thinks one of the modifier keys is depressed. Try pressing and releasing each of left and right Ctrl, Shift, Alt, Windows Key and Context Menu key (between right Ctrl and right Alt).

2. If that doesn't make any difference, can you send us a diagnostic report so we can check on potentially conflicting applications? See http://tavultesoft.com/kb?id=40 for details.
