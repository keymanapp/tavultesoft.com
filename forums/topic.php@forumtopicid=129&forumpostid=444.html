Tavultesoft Staff
Hi Blane,

The version 6.0 keyboard should work fine in Keyman Desktop 7.0. To extract it, go the following folder on your computer when Keyman 6.0 is running with the Bribri keyboard installed:

C:Documents and SettingsAll UsersApplication DataTavultesoftKeymanKeyboard

There should be a subfolder for each installed keyboard -- one should have the Bribri keyboard in it. If it was a package, it will be in the "_Package" subfolder. The keyboard file itself will have a .KMX extension.

Once you've located the Bribri keyboard, copy it to another more easily remembered location. Then open Keyman Desktop 7.0 Configuration, click Install Keyboard, and browse to the Bribri keyboard. Click Open to install it.


Kind regards,

Paul Durdin
