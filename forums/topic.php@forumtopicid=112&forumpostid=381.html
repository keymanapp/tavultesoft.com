Hi Paul,
I found a fix so I thought I'd post it back here for anyone else following behind:

"What you will need to do is re-register the following files using the regsvr32 command (ie. regsvr32 filename.fil) in the command prompt : softpub.dll, wintrust.dll, initpki.dll, ssenh.dll, saenh.dll, gpkcsp.dll, sccbase.dll, slbcsp.dll, mssip32.dll, cryptdlg.dll
Note that a few of those files won't be found in windows 98 but for windows xp."
Source: http://www.geekstogo.com/forum/lofiversion/index.php/t25884.html

Thanks to Google the Oracle!

This fixed Windows Update and Keyman installation.[/url]
