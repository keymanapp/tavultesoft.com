  1. Make sure you have the latest version of Keyman Desktop installed (9.0.477.0, see Support tab in Keyman Configuration)
  2. Install the keyboard from http://keymankeyboards.com/?id=362
  3. Then select the keyboard from the Keyman icon or from the Windows language selector


If you continue to have trouble, send us a diagnostic report following the steps at http://help.keyman.com/kb/40
