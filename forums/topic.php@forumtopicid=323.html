# Configuring LangPlus® TSE Keyboards with Microsoft Office Word   2011-12-27 16:12:56.227
Pradeeka Seneviratne
This configuration helps you to type Sinhala and Tamil more correctly and easily with Microsoft Office Word.

Microsoft Office 2003
In Office Word 2003, by default "Use sequence checking" option is disabled.
If the type checks sequence is enabled then you won’t be able to type the key sequence properly.

How to disable type check sequence
1. Open Office Word 2003
2. In the main menu click Tools -> Options.
3. An option window will open, click the Complex Script Tab
4. There you will find a check box "Use sequence checking".
5. Uncheck this option.
6. Restart Word Application


Microsoft Office 2007
In Office Word 2007, by default the Use sequence checking is disabled.

1. If it is enabled then you can disable by right clicking on one of the menu item.
2. You will see "Customize Quick Access Toolbar", click on that sub menu. Word options window will open.
3. Click Advanced Tab on the right side of the window.
4. Under Editing options you will find the Use sequence checking check box.
5. Uncheck it.
6. Restart Word Application.
