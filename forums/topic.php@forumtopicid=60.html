# RE: how to create a ansi phonetic key board   2006-08-09 09:51:08.667
Marc Durdin
Tavultesoft Staff
You can use character values in four different ways, for either ANSI or Unicode (although U+xxxx is strictly speaking only really for Unicode):

+ 'a' > d192 c decimal value 192 = 0xC0
+ 'b' > xC3 c hexadecimal value 0xC3 = 195
+ 'c' > 306 c octal value 306 = 198
+ 'd' > U+00C9 c hexadecimal value 0xC9 = 201

We recommend either (a) or (b) above for non-Unicode values (octal is only supported for legacy reasons!)
