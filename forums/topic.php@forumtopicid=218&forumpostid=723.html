# Eclipse SWT and Keyman...   2009-02-06 09:02:10.037
LDS Church
Are there known issues with Keyman and SWT based applications like Eclipse? One of our engineers did some testing with the current Eclipse 3.4 and these are the results:

Some CTRL characters do not work at all, such as CTRL+c and CTR+s.

Ctrl+Shift characters only work if Shift is held down first, then Ctrl, then the key is typed.
Alt+Shift characters only work if Shift is held down first, then Alt, then the key is typed.
Ctrl+Alt+Shift characters only work in the order Ctrl then Alt then Shift then key, or Alt then Ctrl then Shift then key.
