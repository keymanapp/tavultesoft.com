# RE: keyman developer issues   2013-06-06 09:47:35.217
Jeremy Kirkham
Tavultesoft Staff
The reason this is occuring is that the SHIFT key is overriding the Numlock - this is built into the hardware and cannot be changed.

Example: With Numlock on, [SHIFT K_NP7] will activate home.
With Numlock off, [SHIFT K_NP7] will activate 7.

We advise not using SHIFT with the numberpad because of this reason.
