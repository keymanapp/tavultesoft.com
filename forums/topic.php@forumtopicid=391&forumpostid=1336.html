Keyman Developer is an application for developing keyboards for individual languages, multiple languages, or lingusitic (special character) use.

Please see http://keyman.com/developer/ for more information. Note that Keyman Developer 9 beta, which can be freely downloaded and used, is fully functional.

(Keyman Developer 8 requires a license to use beyond the 30-day trial period, or to distribute keyboards developed using that version.)
