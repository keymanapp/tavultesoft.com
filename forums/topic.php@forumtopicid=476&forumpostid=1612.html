# RE: Not able to make a statement like this   2016-12-05 20:50:34.067
Marc Durdin
Tavultesoft Staff
Yes, that's not quite correct. You'd need to include [SHIFT ALT] in each of the entries in the store, e.g.

store(shift_alt_digits) [SHIFT ALT '1'] [SHIFT ALT '2'] c ... etc ...

Or

store(shift_alt_digits) [SHIFT ALT K_1] [SHIFT ALT K_2] c ...
