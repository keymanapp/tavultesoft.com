# Can't link keyboard to language   2007-01-26 06:39:42.340
Bruce Button
I'm using the evaluation version of Keyman Developer and Keyman 6.2 with Windows SP2 and trying to link specific keyboards to specific languages.

The "recommended method" in the help file says that one should use the Language tab on Keyman configuration. However, when I go there I am unable to change the keyboards for the different languages because all of them tell me, "You can't associate a Keyman keyboard with this language because it is an IME." The other method - working through the Control Panel - also doesn't change to the required keyboard when changing to the language.

Can someone help me?
# RE: Can't link keyboard to language   2007-01-26 09:42:50.997
Tavultesoft
Tavultesoft Staff
Hi Bruce,

This error message is due to the Windows languages option "Extend support of advanced text services to all programs". With this option enabled (it's in Control Panel->Regional Settings->Languages->Details->Advanced), Keyman will not let you associate a keyboard with any language.

Windows Vista has an equivalent option constantly enabled, but it doesn't appear to cause the same issue.


Kind regards,

Paul Durdin
