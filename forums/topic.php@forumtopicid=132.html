Hi,

We don't have an immediate solution for the issue, unfortunately, but we'll be following it up with Microsoft. It appears that Excel 2007 is capturing all the keyboard shortcuts, which is a change from earlier versions.

If you're using Keyman Desktop Professional 7.0, you can use the character map to enter the characters with diacritics (rather than the Office Insert Symbol tool).

The other option is to try one of the other Sanskrit keyboards which don't use the Alt or Ctrl keys, or use Keyman Developer to create such a keyboard.

When we have a better solution we will contact you.
