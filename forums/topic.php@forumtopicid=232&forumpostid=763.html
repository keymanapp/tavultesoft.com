# RE: Korean(!) language arbitrarily added   2009-06-04 11:04:58.233
Marc Durdin
Tavultesoft Staff
We have observed this behaviour on one other system but have not yet determined the cause. We are still investigating the problem.

Automatic language configuration with Keyman only occurs if the keyboard layout includes a language code. It is not recommended to disable the automatic language configuration as it makes installation of the keyboard far more tedious and error-prone for the average user.
