# RE: Compatibility with French Keyboard   2007-11-28 10:46:02.747
Marc Durdin
Tavultesoft Staff
Jenni,

The solution is to add the other language, but use French as the keyboard layout for the language. For instance in XP:

1. Go to Control Panel, Regional and Language Options, Languages, Details, and click Add
2. Select Afrikaans in the Input language box
3. In the Keyboard layout/IME box, select French
4. Click OK, OK, OK

The Afrikaans language will be installed with the AZERTY layout.
