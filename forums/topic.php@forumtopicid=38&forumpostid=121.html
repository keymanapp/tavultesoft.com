This is not done by Keyman but by the application you are using. Word is the most common culprit. What is happening is that the font you have selected does not contain the letters you are typing. So Word tries to be clever and uses a technology called "font linking" to select a font which does contain those characters. Unfortunately, in this case, the font is chooses is Tahoma.

The simplest way to avoid this is to choose a font which contains all the characters you are using.
