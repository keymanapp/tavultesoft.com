# RE: Some Tamil Letters are not display in Onscreen Keyboard - Keyman 8   2011-07-10 21:47:50.797
Mr Pradeeka Seneviratne
Hi,

I have solved the above problem and these are the steps.

1. In 'Design an on-screen keyboard' in Keyman Developer 8, simply delete the loaded character (loaded by Fill from layput button)in 'Text' text box.

2. Drag the character again from the Character Map (change the insert mode to character in Character Map )

3. Compile the keyboard

Now in keyman Developer 8 '?' appears correctly on the on-screen keyboard key.

But in Keyman Developer 7.1 no need to apply above steps.
