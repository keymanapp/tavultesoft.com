The process is C:\Program Files\Replay Media Catcher\FLVSrvc.exe. Note that it isn't enough to terminate the process in Task Manager, because the (potentially) conflicting DLL will remain in memory in other processes. You need to prevent it from starting with Windows.

A good tool for managing applications that start with Windows is Autoruns from http://technet.microsoft.com/en-us/sysinternals/bb963902.aspx
