Hi Torsten,

Try uninstalling Keyman, rebooting the computer, downloading the installer again and re-installing from scratch. If you still get the same error, please look for a file called 'ExcMagic.Log' in either of the following folders:

C:\Program Files\Tavultesoft\Keyman Desktop Professional 7.0
C:\Program Files\Common Files\Tavultesoft\Keyman Engine 7.0

If that log file exists, please email it to us at support@tavultesoft.com so we can determine the cause of the error!
