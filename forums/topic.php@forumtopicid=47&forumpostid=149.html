Hi Thomas,

I've just verified that the Home Edition installer works, both downloading it and running it directly off the website.

It sounds like the file was not downloaded properly the first time you tried -- the second time it may be using the same cached file.

Try clearing out your browser cache and redownloading the file.

A second possibility, but less likely, is that anti-virus software is preventing it from running properly. Try disabling any anti-virus software before running the installer.
