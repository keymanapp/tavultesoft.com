Which version of Keyman are you using?

Where can this keyboard be downloaded? It looks like it is trying to set a registry setting for Surrogate Pair support in Windows; this is not usually required and I am surprised that it would be needed for a TAM 8 bit keyboard.

The reason the install is failing is that it requires Administrator access to install. You can start Keyman Configuration as an Administrator by:

1. Start Explorer, and locate C:\Program Files\Tavultesoft\Keyman Desktop Professional 7.0.
2. Hold SHIFT down, and right-click on kmshell.exe.
3. Select "Run as administrator".
