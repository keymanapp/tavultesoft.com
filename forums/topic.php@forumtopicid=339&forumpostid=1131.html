OK, I'm not sure what the Private Character Editor is? A font describes the shapes of the characters - both Unicode-defined characters and Private Use characters. If you are adding characters to the Private Use Area, that necessarily requires a custom font to go with it in order for the shape of the characters to be viewable.
# RE: New Glyphs with no unicode   2013-03-26 21:58:17.700
Marc Durdin
Tavultesoft Staff
So I've just looked it up, and this is a very limited tool intended really only for use on your own computer. It hacks in bitmap character shapes (not scaleable) to the display renderer; Keyman Developer won't support this in Character Map because the characters are not enumerable from the font resource itself.

In order to distribute a solution with PUA characters, you'll need a font tool such as FontLab to create your own font.

# RE: New Glyphs with no unicode   2013-03-28 08:28:07.343
Marc Durdin
Tavultesoft Staff
If you create a custom font and install it in Windows, it will appear in the font list in Character Map in the same way as any other font.

Note: if you are running Windows 7 or Windows 8, you may want to disable the option "Hide fonts based on language settings" in the Font settings control panel.
