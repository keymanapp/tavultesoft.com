Hi,

What you are trying to to is not actually possible with Keyman Desktop (or any other keyboard mapping tool that we know of), since you cannot use an external IME, such as Google IME, then a keyboard rule to modify the output of the IME, as the IME output goes straight to the application, not into the keyboard input queue.

While in theory it is possible to create a special purpose IME keyboard to do that with Keyman Desktop, it would be quite difficult to design and would take a lot of time to develop.

If you are using an application with macros or scripting, it might be possible to trap and modify the output of the IME using macro or scripting code, but discussion of that is really outside the scope of this forum.
