# RE: Easiest way to type Latin?   2010-03-10 10:33:47.850
Tavultesoft
Tavultesoft Staff
Macron is used widely in the Pacific but not so much in other areas, so it tends not to be available in European-focused keyboards. A couple of ways to solve this:

1. Use an existing pan-Latin keyboard that does support macron and acute. I believe both European Latin (http://eurolatin.keymankeyboards.com/) and BU Keyboard (http://keymankeyboards.com/391) support macron and acute.

2. Create your own keyboard in Keyman Developer: http://www.tavultesoft.com/keymandev/
