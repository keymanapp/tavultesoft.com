Hi Episteme,

If you need accents for the French words, the best way would probably be to download the European Latin keyboard for Keyman. Then set simple hotkeys for the Farsi Unicode keyboard and the European Latin so you can switch between them. You will also have to switch between Farsi and English Windows layouts.

Does that make sense?

With Keyman 6.2 Standard Edition or Keyman Desktop Professional 7.0 you can also associate the Keyman keyboards with the Windows layouts. So instead of enabling both the Farsi Windows layout and the Farsi Keyman keyboard separately, you can enable them both with a single hotkey. This would make switching between Farsi and English/French simpler as well.
