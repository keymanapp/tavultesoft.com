# Sinhala (Wijesekara) Layout Conflicts with LangPlus TSE Keyboards Layout   2011-12-27 16:55:18.843
LangPlus TSE Keyboards
Please 'exit' or disable any previously installed keyboard layout drivers from Windows System trey (Notification Area). Otherwise LangPlus TSE Keyboards will produce wrong letter output.

Known Solutions:
'Exit'/disable two keyboard layout drivers installed with the Sinhala/Tamil kit. (Do not Uninstall Sinhala/Tamil kit.)
