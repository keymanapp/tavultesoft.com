# RE: New Glyphs with no unicode   2013-02-21 16:26:20.743
Jeremy Kirkham
Tavultesoft Staff
Hi Rajendra,

There is an area of Unicode known as private use area which should contain the glyphs you have created.

To find the private use area, firstly select the font you wish to use by right clicking on the Character Map and selecting "font". Then you will need to type the greater than symbol > in the "filter by" input box in the Character Map and then scroll to the Private Use Area in the Character Map. You should then be able to drag and drop your glyphs onto the keyboard.

Regards,
Jeremy

# RE: New Glyphs with no unicode   2013-03-26 21:23:18.460
Marc Durdin
Tavultesoft Staff
Keyman Developer will read the private use characters from your font and display them in the Character Map - you don't need to upload or import it specially into Keyman Developer. The only thing you need to do is select your font in the Character Map and enter ">" into the filter box, which tells the Character Map to use show all the characters in the selected font, rather than all the characters in the Unicode Standard.
