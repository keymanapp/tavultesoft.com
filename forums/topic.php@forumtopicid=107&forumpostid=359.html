Hi Nick,

To associate the Ezra keyboard with the Windows Hebrew input language:

1) Make sure you have the Windows Hebrew input language installed (from Control Panel->Regional Settings).
2) Open Keyman Configuration.
3) Select the Languages tab.
4) For the Hebrew item in the Langage column, select Ezra in the Keyman Keyboard column.

You should then be able to use the Ezra keyboard when you select Hebrew from the Windows Language Bar.

Paul Durdin

Hi Nick,

With Keyman Home Edition you can't associate a keyboard with a Windows Input Language. You can still use the Ezra keyboard by enabling it from Keyman (left-click on the Keyman icon and select it from the menu, or set up a hotkey for the keyboard in Keyman Configuration), but you won't be able to enable it with the Windows Language Bar.

This limitation does not exist in Keyman Standard Edition; I didn't realise you were using the Home Edition!


Kind regards,

Paul Durdin
