The home edition does not support the visual keyboard; you would need to upgrade to the Standard Edition in order to do that. Further details about the differences between the editions are available at:
http://www.tavultesoft.com/keyman/editions.php

You can select Keyman keyboards from the list of keyboards available for a language. In Control Panel, Regional and Language Options, click the Languages tab, and then the Details button. Click the Add button in the Text Services and Input Languages dialog that appears, select the language you wish to install, and then select the Keyman keyboard from the list below that. Advanced associations with languages are available in the Standard Edition.
