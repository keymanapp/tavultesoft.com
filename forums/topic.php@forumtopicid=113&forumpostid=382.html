Hi Wayne,

This issue has now been fixed and is available in the latest version of Keyman Desktop Light (build 7.0.232.0).

You should be able to update your copy using the automatic updater in Keyman Desktop:
1) Open Keyman Configuration
2) Go to the Support Tab.
3) Click 'Check for Updates'. This should download and install the 7.0.232.0 patch and upgrade your copy.

If the above fails for any reason, you can download the full installer from:
http://www.tavultesoft.com/70/download.php
