Hi Xavier,

When Keyman is installed and running, you can see a little 'K' key icon in the bottom right of the screen. Load up Wordpad (or another application), then click the 'K' key icon, select IPA from the list of keyboards installed, and you should be able to just start using it.

Hi Xavier,

As far as I know, the SIL IPA keyboard does not include capital letters - only lower case. In order to see the IPA symbols, you must make sure that you select the correct font (SILDoulos IPA93 for the older, non-Unicode keyboards, or Doulos SIL I think for the newer, Unicode IPA keyboard).

See http://scripts.sil.org/cms/scripts/page.php?site_id=nrsi&amp;item_id=IPAhome for notes on whether or not to use a Unicode IPA keyboard!
