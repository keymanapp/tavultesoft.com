# RE: Keyman 7.1 and Windows 7 64bit   2010-08-11 09:35:37.737
Tavultesoft
Tavultesoft Staff
TW: This behaviour is logical when you look into the underlying Windows x64 support. Both Notepad and Explorer are 64-bit applications in Windows 7 x64. Keyman Desktop 7.1 (and the core engine, which we call Keyman Engine) is a 32-bit application and cannot hook into any 64-bit applications, so logically then no Keyman input or keyboard activation hotkeys can occur.

There are 32-bit versions of most Windows applets, including Notepad, in C:\Windows\SysWow64, with which Keyman Engine 7.1 works fine. Yes, that folder name is rather unintuitive -- it contains all the 32-bit applications, and the 64-bit versions are in System32!

The version 8.0 beta downloadable at http://www.tavultesoft.com/beta/ fully supports 64-bit applications -- it includes a 64-bit version of Keyman Engine that integrates transparently with the 32-bit version. We are working hard on the full version 8.0 release.
