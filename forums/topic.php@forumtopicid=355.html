Yes, this is a known issue with Word -- it often thinks it knows better than the user does which font should be selected for a given script. As far as we have been able to determine, there is no reliable workaround in Word itself, which is frustrating.

However, you may find that Keyman 9, which uses a different method of inserting characters into Word, gives you a better result -- this is available at http://tavultesoft.com/beta/ as a prerelease product right now.
