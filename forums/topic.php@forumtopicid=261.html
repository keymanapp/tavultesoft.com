The uninstaller for Keyman Developer can be found in the standard Windows Control Panel "Add/Remove Programs" applet -- the normal place to uninstall or repair any software installation. Microsoft guidelines state that software should not include an uninstall link in the Start Menu (http://msdn.microsoft.com/en-us/library/aa511447.aspx).

You should never delete files out of the Program Files folder directly, for any software -- this can corrupt your software installation (as you found). The "Cached Installers" folder contains a cache of the original installation MSI file for Keyman Developer so that uninstallation, upgrades, repairs and changes to the installation can be made without needing the original installation files. This is not intended for the end user to access directly -- you should be using the Windows Control Panel Add/Remove Programs applet to maintain your software installations.
